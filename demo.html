📚 BookmarkHub Manager - 演示页面

🚀 快速开始

这是一个功能完整的书签管理工具，支持从 GitHub Gist 同步 BookmarkHub 扩展的书签数据。

🔗 打开应用 📋 查看演示数据

🔄 GitHub 同步

从 GitHub Gist 获取和同步书签数据，支持自动同步和手动同步。

🔍 智能搜索

实时搜索书签标题和 URL，支持模糊匹配和高亮显示。

📁 文件夹管理

创建、重命名、删除文件夹，支持拖拽排序和层级管理。

🔒 隐私保护

支持隐藏敏感书签，可设置密码保护隐藏内容。

⚡ 批量操作

支持批量删除、移动、修改书签，提高管理效率。

🔧 实用工具

链接有效性检查、撤销操作、数据导入导出等功能。

📋 使用步骤

准备 GitHub 凭据

在 GitHub 创建 Personal Access Token，确保具有 gist 权限。

获取 Gist ID

创建包含 BookmarkHub 数据的 Gist，或使用现有的 Gist ID。

配置应用

在应用设置中输入 GitHub Token 和 Gist ID。

同步数据

点击同步按钮获取书签数据，开始使用所有功能。

🛠 技术特性

  • 纯前端应用：无需服务器，可部署到任何静态托管服务
  • IndexedDB 存储：本地数据持久化，支持离线使用
  • 响应式设计：适配桌面和移动设备
  • 模块化架构：清晰的代码结构，易于维护和扩展
  • 现代 Web API：使用最新的浏览器功能

⚡ 快捷键

  • Ctrl+F - 聚焦搜索框
  • Ctrl+Z - 撤销上一个操作
  • Ctrl+S - 同步书签数据
  • Esc - 关闭当前对话框
